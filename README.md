# API Sample

## Description

This is a sample of API with NodeJS and Express.

## Requirements

node : v18.6.0
</br>
bash : npm run dev
</br>

To kill if port 3000 is already used :
```bash
kill port 3000 : kill -9 $(lsof -t -i:3000)
```

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev
```
